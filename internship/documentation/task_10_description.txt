При выполнении данного задания я работала в классе TravelCalculatePremiumRequest. В названном классе я создала
дефолтный конструктор, конструктор для задания начальных значений всем свойствам, геттеры и сеттеры для всех
свойств класса.
После выполнения каждого шага я коммитила сделанные изменения в локальный Git. При работе с локальным Git
 я использовала следующие команды:
git status
git commit -a -m "message"
git log
Также все сделанные изменения были запушены в глобальный репозиторий (с использованием опции rebase):
git pull —rebase origin main
git push origin main
Разница между git merge и git rebase заключается в том, что git merge создает новый коммит слияния,
который объединяет изменения из обеих веток, а git rebase перемещает коммиты из одной ветки на вершину другой ветки.
