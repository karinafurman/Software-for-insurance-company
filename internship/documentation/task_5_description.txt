В ходе выполнения этого задания я познакомилась с артефактами,
которые создались в процессе сборки приложения. Артефакты, созданные Gradle
в процессе сборки, находятся в папке build проекта.
classes — bytecode откомпилированных классов
classes —> java —> main — bytecode продакшн классов
classes — java —> test —> bytecode тестовых классов
generated — сгенерированный при компиляции код
libs — jar-файл, собранный из продакшн кода
reports — сгенерированный отчет по тестам
resources — файлы ресурсов
test-results — результаты выполнения тестов
