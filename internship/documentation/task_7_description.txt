P.S. Можно ли запустить две копии приложения одновременно?
Да, можно если приложение не имеет ограничений, запрещающих параллельный запуск.
Для того, чтобы это сделать, нужно перейти в директории, содержащие скомпилированный .jar файл или
классы приложения одновременно в двух терминалах, запустить приложение в каждом терминале с помощью команды java:
java -jar [наименование приложения].jar
или
java [наименование main-класса]
