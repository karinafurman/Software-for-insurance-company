В ходе выполнения данного задания я применила подход, описанный ментором для создания mock объектов
и внедрения их качестве зависимости в классе в тестовом классе travel TravelCalculatePremiumServiceImplTest
Аннотация @ExtendWith включает поддержку Mockito в JUnit 5
Аннотация @Mock создаёт mock объект
Аннотация @InjectMocks создаёт тестируемые объекты внедряет в него все mock-зависимости
По сути создаётся тестируемый сервис @InjectMocks, куда автоматически подставляются все mock-зависимости
Во всех остальных тестах класса TravelCalculatePremiumServiceImplTest было удалено создание локальной
переменной request, поскольку переменная request создавалась в методе setUp() с аннотацией @BeforeEach.
Наличие в тестах локальных переменных request не позволялo программе билдиться, поскольку для локальных
переменных request не были настроены моки
